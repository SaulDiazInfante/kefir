## Report Changes

Main report:
`docs/kefir_model_comparison_report.tex`

References:
`docs/kefir_model_comparison_references.bib`

PDF rebuilt:
`docs/kefir_model_comparison_report.pdf`

Report now includes:

- Five-model abstract: classical logistic ODE, Neural ODE, Neural SDE,
  deterministic logistic PINN, stochastic logistic PINN/SDE.
- Data/preprocessing clarification that Neural ODE/SDE use normalized response,
  while logistic PINNs use original response scale with losses divided by
  `s_y^2`.
- Full logistic PINN inverse-model section:
  - unknowns `eta, r, K` for deterministic PINN;
  - unknowns `eta, r, K, sigma` for stochastic PINN/SDE;
  - trajectory network architecture: input `tau`, 3 hidden layers, 32 units
    each, `tanh`, softplus positive output;
  - parameter constraints via softplus;
  - deterministic data, initial, and physics losses;
  - stochastic Euler-Maruyama transition likelihood;
  - optimization loops for classical logistic ODE, deterministic PINN, and
    stochastic PINN/SDE.
- Updated numerical results tables with PINN metrics and parameter estimates.
- Added figures:
  - `water_kefir_logistic_pinn_comparison.png`
  - `water_kefir_logistic_pinn_losses.png`
- Added reproducibility command for:
  `kefir-logistic-pinn-compare --config logistic_pinn_config_reference.json`
- Added PINN citation:
  `raissi2019physics`
